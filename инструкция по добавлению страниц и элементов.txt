========== Добавление кликабельных элементов ==========

1.  Добавить тег в main по шаблону:

    <div class="device" data-name="[НАЗВАНИЕ]"></div>

2. В javaScript файле в объект 'deviceList' (в самом низу) добавить информацию о детали ПК по шаблону:

DEVICE: { (вместо DEVICE название которое ДОЛЖНО СОВПАДАТЬ с названием из атрибута 'data-name' в HTML)
    title: 'Деталь', <--- Название детали
    descr: 'Деталь такая-то, делает то-то', <--- Описание детали
    url: 'https://ru.wikipedia.org', <--- Ссылка на статью в Интернете о детали,
    imgSrc: '[ССЫЛКА НА КАРТИНКУ]', <--- Картинка-иллюстрация
    detail: true/false, <--- Есть ли детальный вид (показывать ли соотв. кнопку)
    detailedView: '[ССЫЛКА НА HTML ФАЙЛ]', <--- ссылка на приближённый вид детали
    removable: true/false, <--- Можно ли убрать (показывать ли соотв. кнопку)
    remove: function() {...} <--- функция, которая будет выполняться при нажатии кнопки 'убрать'
}

3. Самостоятельно прописать все стили
    Пример:
        .device[data-name="motherBoard"] {
            width: 58%;
            height: 58%;
            position: relative;
            left: 16%;
            top: 17%;
